Verification Framework Description

R-T6: The Car shall only stop at Floors for which there are pending calls.
Transition #
 Conditions
RT6-1
 mDriveSpeed <= LevelSpeed AND  any mHallCall[currenfloor, b] is true or mCallCall[currentfloor, b] is true
RT6-2
 mDriveSpeed > LevelSpeed
RT6-2
 mDriveSpeed <= LevelSpeed AND  all mHallCall[currenfloor, b] or mCallCall[currentfloor, b] are not true
RT6-4
 mDriveSpeed > LevelSpeed

write up: R-T6 is violated if and only if car stops at a floor at which there is no call RT6-3. The monitor records car's state change from above level speed to below level speed RT6-1 (attempting to stop). A warning is thrown when there is no pending call while making transition RT6-3.
 

R-T7: The Car shall only open Doors at Hallways for which there are pending calls.



Transition #
 Conditions
RT7-1
 any mDoorOpen[b, r] is true AND any mHallCall[currenfloor, b] is true or mCallCall[currentfloor, b] is true
RT7-2
 mDoorOpen[b, r] is false
RT7-2
 any mDoorOpen[b, r] is true AND all mHallCall[currenfloor, b] or mCallCall[currentfloor, b] are not true
RT7-4
 mDoorOpen[b, r] is false

write up: R-T7 is violated if and only if the door opens at a hallway in which there is no pending calls. The monitor records door's state from close to open. A warning is thrown when there is no pending calls while making transition RT7-4.

R-T8.1: If any door is open at a hallway and there are any pending calls at any other floor(s), a Car Lantern shall turn on.



Transition #
 Conditions
RT8_1-1
 any mDoorMotor[b, r] is open AND (any mHallCall[otherFloor, b]) or mCarCall[otherFloor, b] is true ) AND (CarLantern[up] or CarLantern[down] is true)
RT8_1-2
 any mDoorMotor[b, r] is open AND (any mHallCall[otherFloor, b]) or mCarCall[otherFloor, b] is true ) AND (CarLantern[up] and CarLantern[down] are false )
RT8_1-3
 all mDoorOpen[b, r] are false 
RT8_1-4
 all mDoorOpen[b, r] are false

write up: R-T8.1 is violated if and only both lantern is off while any door is open and there are pending calls. The monitor records lantern's state while the door is open. A warning is thrown when there is pending calls at other floor and both lantern is not lit on while making transition RT8_1-2.

R-T8.2: If one of the car lanterns is lit, the direction indicated shall not change while the doors are open.



Transition #
 Conditions
RT8_2-1
 any mDoorOpen[b,r] is true and mCarLantern[up] is true
RT8_2-2
 any mDoorOpen[b,r] is true and mCarLantern[down] is true
RT8_2-3
 all mDoorOpen[b,r] are false
RT8_2-4
 any mDoorOpen[b,r] is true and mCarLantern[up] is false
RT8_2-5 all mDoorOpen[b,r] are false
RT8_2-6 any mDoorOpen[b,r] is true and mCarLantern[down] is false
RT8_2-7 all mDoorOpen[b,r] are false

write up: R-T8.2 is violated if and only if the lantern lit on switched from one to another while the door is open. The monitor records lantern's state while any door is open. A warning is thrown when any changes are made in lantern while doors are open while making transition RT8_2-3 & RT8_2-6.

R-T8.3: If one of the car lanterns is lit, the car shall service any calls in that direction first.



Transition #
 Conditions
RT8_3-1
 all mDoorClosed[b,r] are false and mCarLantern[down] is true
RT8_3-2
 all mDoorClosed[b,r] are false and mCarLantern[up] is true
RT8_3-3
 all mDoorClosed[b,r] are false and mCarLantern[up] is false
RT8_3-4
 all mDoorClosed[b,r] are false and mCarLantern[up] is true and desiredFloor < currentFloor and (mHallCall(higherFloor, b) is true or mCarCall(higherFloor, b) is true )
RT8_3-5 all mDoorClosed[b,r] are false and mCarLantern[down] is false
RT8_3-6 all mDoorClosed[b,r] are false and mCarLantern[down] is true and desiredFloor > currentFloor and (mHallCall(lowerFloor, b) is true or mCarCall(lowerFloor, b) is true )
RT8_3-7 all mDoorClosed[b,r] are false

write up: R-T8.3 is violated if and only if the car services one direction first while lantern for other direction is already lit on. The monitor records dispatcher's desired floor while the door is and one lantern is lit on. A warning is thrown when car's desired floor is not on the same direction as the lantern indicated while making transition RT8_3-4 & RT8_3-6.

R-T9: The Drive shall be commanded to fast speed to the maximum degree practicable.




Transition #
 Conditions
RT9-1
 Currentloor != None and fastspeed is reached and mDriveSpeed == 0
RT9-2
 Currentloor != None and fastspeed is not reached and mDriveSpeed == 0
RT9-2
 mDriveSpeed != 0
RT9-4
 mDriveSpeed != 0

write up: R-T9 is violated when car is moving between two floors and fast speed is never reached. The monitor records drive speed and checks if it makes transition from slow speed to fast speed. A warning is thrown when the drive remains when moving to next floor.
