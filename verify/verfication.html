Verification Framework Description

R-T6: The Car shall only stop at Floors for which there are pending calls.
Transition #
 Conditions
RT6-1
 mDriveSpeed <= LevelSpeed AND  any mHallCall[currenfloor, b] is true or mCallCall[currentfloor, b] is true
RT6-2
 mDriveSpeed > LevelSpeed
RT6-2
 mDriveSpeed <= LevelSpeed AND  all mHallCall[currenfloor, b] or mCallCall[currentfloor, b] are not true
RT6-4
 mDriveSpeed > LevelSpeed




R-T7: The Car shall only open Doors at Hallways for which there are pending calls.



Transition #
 Conditions
RT7-1
 any mDoorOpen[b, r] is true AND any mHallCall[currenfloor, b] is true or mCallCall[currentfloor, b] is true
RT7-2
 mDoorOpen[b, r] is false
RT7-2
 any mDoorOpen[b, r] is true AND all mHallCall[currenfloor, b] or mCallCall[currentfloor, b] are not true
RT7-4
 mDoorOpen[b, r] is false




R-T8.1: If any door is open at a hallway and there are any pending calls at any other floor(s), a Car Lantern shall turn on.



Transition #
 Conditions
RT8_1-1
 any mDoorMotor[b, r] is open AND (any mHallCall[otherFloor, b]) or mCarCall[otherFloor, b] is true ) AND (CarLantern[up] or CarLantern[down] is true)
RT8_1-2
 any mDoorMotor[b, r] is open AND (any mHallCall[otherFloor, b]) or mCarCall[otherFloor, b] is true ) AND (CarLantern[up] and CarLantern[down] are false )
RT8_1-3
 all mDoorOpen[b, r] are false 
RT8_1-4
 all mDoorOpen[b, r] are false



R-T8.2: If one of the car lanterns is lit, the direction indicated shall not change while the doors are open.



Transition #
 Conditions
RT8_2-1
 any mDoorOpen[b,r] is true and mCarLantern[up] is true
RT8_2-2
 any mDoorOpen[b,r] is true and mCarLantern[down] is true
RT8_2-3
 all mDoorOpen[b,r] are false
RT8_2-4
 any mDoorOpen[b,r] is true and mCarLantern[up] is false
RT8_2-5 all mDoorOpen[b,r] are false
RT8_2-6 any mDoorOpen[b,r] is true and mCarLantern[down] is false
RT8_2-7 all mDoorOpen[b,r] are false


R-T8.3: If one of the car lanterns is lit, the car shall service any calls in that direction first.



Transition #
 Conditions
RT8_3-1
 all mDoorClosed[b,r] are false and mCarLantern[down] is true
RT8_3-2
 all mDoorClosed[b,r] are false and mCarLantern[up] is true
RT8_3-3
 all mDoorClosed[b,r] are false and mCarLantern[up] is false
RT8_3-4
 all mDoorClosed[b,r] are false and mCarLantern[up] is true and desiredFloor < currentFloor and (mHallCall(higherFloor, b) is true or mCarCall(higherFloor, b) is true )
RT8_3-5 all mDoorClosed[b,r] are false and mCarLantern[down] is false
RT8_3-6 all mDoorClosed[b,r] are false and mCarLantern[down] is true and desiredFloor > currentFloor and (mHallCall(lowerFloor, b) is true or mCarCall(lowerFloor, b) is true )
RT8_3-7 all mDoorClosed[b,r] are false
