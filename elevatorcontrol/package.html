This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

Links and descriptions to all controllers.

  • CarButtonControl.java - This is controller for Car Button.
  • CarPositionControl.java - This is controller for Car Position Indicator.
  • Dispatcher.java - The dispatcher determine the elevator's future destinations.
  • DoorControl.java - This is door operation controller.
  • DriveControl.java - This is controller for elevator drive.
  • HallButtonContol.java - This is controller for Hall Button.
  • LanternControl.java - This is controller for Car Lantern.

Translator files

Links and descriptions to all message translators used by the controllers.
  • CarCallCanPayloadTranslator.java - CAN translator for messages from CarCall sensors.
  • CarLightCanPayloadTranslator.java - CAN translator for messages from CarLight sensors.
  • DesiredDwellCanPayloadTranslator.java - CAN translator for mDesiredDwell
  • DesiredFloorCanPayloadTranslator.java - CAN translator for mDesiredFloor
  • DriveCommandCanPayloadTranslator.java - CAN translator for mDriveCommand messages
  • DriveSpeedCanPayloadTranslator.java - CAN translator for mDriveSpeed messages
  • HallCallCanPayloadTranslator.java - CAN translator for messages from HallCall sensors.

Utility files

  • Utility.java - This class provides some utility classes that is useful in more than one spot.
