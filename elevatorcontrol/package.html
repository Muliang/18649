This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

Links and descriptions to all controllers.

  • CarButtonControl.java - Car Button controller switches on CarLight whenever user makes a car call.
  • CarPositionControl.java - Car Position controller displays current floor.
  • Dispatcher.java - Dispatcher determines the elevator's future destination.
  • DoorControl.java - Door controller operates door opening/closing/nudging.
  • DriveControl.java - Drive controller determines the elevator's moving speed with direction.
  • HallButtonContol.java - Hall Button controller switches on HallLight whenever user makes a hall call.
  • LanternControl.java - Lantern controller displays elevator's immediate moving direction.

Translator files

Links and descriptions to all message translators used by the controllers.
  • ByteBooleanCanPayloadTranslator.java CAN translator which can be extended by CarCallCanPayloadTranslator and HallCallCanPayloadTranslator.
  • CarCallCanPayloadTranslator.java - CAN translator for messages from CarCall sensors.
  • CarLightCanPayloadTranslator.java - CAN translator for messages from CarLight sensors.
  • DesiredDwellCanPayloadTranslator.java - CAN translator for mDesiredDwell messages.
  • DesiredFloorCanPayloadTranslator.java - CAN translator for mDesiredFloor messages.
  • DoorMotorCommandCanPayloadTranslator.java - CAN translator for mDoorMotor messages.
  • DriveCommandCanPayloadTranslator.java - CAN translator for mDriveCommand messages.
  • DriveSpeedCanPayloadTranslator.java - CAN translator for mDriveSpeed messages.
  • HallCallCanPayloadTranslator.java - CAN translator for messages from HallCall sensors.

Utility files

  • Utility.java - This class provides some utility classes that is useful in more than one spot. 
  • Proj7RuntimeMonitor.java - performance monitor for Project 7.
  • Proj11RuntimeMonitor.java - high level requirements monitor for Project 11.
